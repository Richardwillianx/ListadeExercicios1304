Lista de exercícios com JavaScript

Tendo como entrada a altura e o sexo (codificado da seguinte forma: 1 para sexo feminino e 2 para sexo masculino) de uma pessoa, construa um programa que calcule e imprima seu peso ideal, utilizando as seguintes fórmulas:

- para homens: (72.7 * Altura) – 58
- para mulheres: (62.1 * Altura) – 44.7

Exemplo de dados:
altura = 1.88
sexo = 2
